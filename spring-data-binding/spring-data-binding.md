## 注解
| 注解             | 用途                                                |
|:---------------|:--------------------------------------------------|
| @RequestParam  | Content-Type: application/x-www-form-urlencoded   |
| @RequestBody   | 非 Content-Type: application/x-www-form-urlencoded |
| @PathVariable  | 请求路径占位符，便于 RESTFul API                            |
| @RequestHeader | 请求头                                               |
| @CookieValue   | Cookie                                            |
| @InitBinder    | 1.相同属性名<br/>2. 类型转换                               |
